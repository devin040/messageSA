# messageSA
A side project for experimenting with some NLP, classification, sentiment analysis

# Dependencies
This project uses:
  - NLTK (tokenizing, stemming)
  - Spacy (tokenizing, stemming)
  - SciKit (classification, metrics)
  - Yellowbrick (viz)
